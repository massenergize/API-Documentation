# User Profile

### /users.import

<Highlight level={1}>Admins Only</Highlight>

if `csv` argument is given:
- Required: `community_id:int`, `csv:file`, `first_name_field:str`, `last_name_field:str`, `email_field:str`
- Optional: `team_id:int`

else:
- Required: `community_id:int`, `names:str_list`, `first_names:str_list`, `last_names:str_list`, `emails:str_list`
- Optional: `team_id:int`

This route is used to import users from either CSV file or given list.
