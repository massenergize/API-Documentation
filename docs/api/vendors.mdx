# Vendors

This page contains all routes related to general actions for vendors.

## Routes

### /vendors.info

- Required: `vendor_id:int` or `id:int`

### /vendors.create

<Highlight level={1}>Admins Only</Highlight>

Expect:
- `key_contact_name:str`
- `key_contact_email:str`
- `onborading_contact_email:str`
- `name:str`
- `email:str`
- `phone_number:str`
- `have_address:bool`
- `is_verified:bool`
- `website:str`
- `is_published:bool`
- `is_approved:bool`
- `communities:listt`
- `service_area_states:str_list`
- `properties_services:str_list`
- `image:str_list`
- `tags:list`
- `location:location`
